# Ultron

## Description

本项目为Ultron系统的配置端<br/>
Ultron服务端暂未开发-.-

基于React实现的组件化的一个配置平台<br/>
通过在界面的配置，来发布上线一个web页面

启动方式：<br/>
首先需要安装依赖
```
$ npm i
$ cd src
$ npm i
$ gulp
$ cd ..
$ npm start
```
现在，就可以看到效果了（目前还是一个灰常简陋的版本，只有俩组件可以使用。。。Container和Text）
```
127.0.0.1:12306
```
